## Comandos git

### init
### status

### add
Adiciona o conteúdo do arquivo ao índice 
```bash 
git add
```
### commit
Registra alterações no repositório
```bash 
git commit
``````
### pull
Atualiza as alterações feitas por si ou por outros coloboradores
```bash
git pull
```
